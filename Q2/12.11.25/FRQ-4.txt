public static int numSymbols (String s) {
    int symbols = s.length() - ((numDigits(s) + numLetters(s)));
    return symbols;
}

public static String passwordStrength(String p) {
    if (p.length() >= 8 && numDigits(p) >= 1 && numLetters(p) >= 1 && numSymbols(p) >= 1) {
        String output = "strong";
        return output;
        //return "strong";
    }

    else if ((p.length() >= 5 && p.length() < 8 && numSymbols(p) >= 1) || p.length() >= 8 && (numDigits(p) == 0 || numLetters(p) == 0 || numSymbols(p) == 0)) {
        String output = "medium";
        return output;
    }

    else {
        String output = "weak";
        return output;
    }
}

notes

- you can return a string directly
    - for instance `return "Strong";` is valid you don't need to put it into a variable

Type 1 FRQs will always have:
- loops (for/while)
- if else
- function call
- parameter passing
- function signature -> as written in the directions
    - for instance `public static String passwordStrength(String p)` should be exactly as it is in the direction

- If you include all of these things you ensure that you get maxium points
    - ie. if you can use a for loop use it
